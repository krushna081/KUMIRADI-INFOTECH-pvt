## Animation Guide

This project includes a dedicated animation framework.

### Included Effects

* Fade In Animations
* Slide Animations
* Hover Lift Effects
* Glow Effects
* Pulse Effects
* Smooth Navigation Transitions
* Card Entrance Animations
* Ripple Effects
* Counter Animations
* Mobile Menu Animations

### Setup

Add:

```html
<link rel="stylesheet" href="css/smooth-transitions.css">
```

and

```html
<script src="js/smooth-animations.js"></script>
```

to enable the complete animation system.

## Installation

### Clone Repository

```bash
git clone https://github.com/krushna081/KUMIRADI-INFOTECH-pvt.git
```

### Open Project

```bash
cd KUMIRADI-INFOTECH-pvt
```

Open any HTML page in your browser or run using a local server.

Example:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
